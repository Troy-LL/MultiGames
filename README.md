# Student Retention Model

A visualization tool designed to track and understand how retention works throughout the student life cycle. 🎓

This project marks the beginning of my journey into data science and analysis. After months of studying the core concepts, I built this model to bridge the gap between theoretical knowledge and practical application, allowing for a deeper look into student behavior and outcomes.

## Core Objective

The primary goal of this repository is to visualize the "student memory map"—how knowledge is retained, where the gaps occur, and how we can better support students through data-driven insights.

## Project History

- **Context**: Visualizing student life cycles.
- **Milestone**: First practical data analysis project after months of self-study.
- **Tech**: React + TypeScript + Vite.

## Getting Started

To run the project locally:

```bash
npm install
npm run dev
```
